Python is an interpreted language 

C or C++ on the other hand and compiled languages and they allow us to observe how each of our instructions interact with the underlying hardware 

Now whether a program comes from an interpreter or a compiled file, its instructions and data must always first be loaded into the main memory before the CPU can run them.

READ OPERATION 
CPU places the address on the address bus signaling the read operation on the control bus, and the system memory responds by placing the required data on the data bus.

WRITE OPERATION 
In case of a write operation the CPU places the data to write on the data bus and the address where that data is to be written is placed on the address bus.
