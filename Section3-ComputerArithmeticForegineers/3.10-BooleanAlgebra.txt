Boolean Algebra is the fundamental concept that powers modern computers, digital electronics and all logic based systems.

In the 19th century some guy named George Boole came up with
A new branch of mathematics to model logical thinking using symbols and equations 

A system where variables have only 2 possible values either True OR False 

This laid the ground work for logic control, control systems, digital logic, and eventually the structure of computer processors

The Boolean Algebra provides a compact and efficient way to represent truth conditions that occur frequently in programming, hardware control and logical automation

Operands are the values on which Boolean Operators act on 

A truth table is a tool used to show how the output of a logic operation changes based on all possible combinations of inputs.

Logic gates are physical electronic components that perform Boolean Operations 

AND = conjunction operator 
OR = disjunction operator 
NOT = negation operator 

Interesting thing we have all seen the NOT gate's symbol 
The triangle in that represents the flow of logic and the circle means invert the input
